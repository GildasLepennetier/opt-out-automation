# opt-out-automation

## introduction

The open-source project that provide a flexible solution to automate the process of opting out of data brokers.

Several projects and solutions exist, for example:

- [justvanish](https://github.com/AnalogJ/justvanish)
- [incogni](https://incogni.com/)
- [deleteme](https://joindeleteme.com/)

## requirements and installation

[Python](https://www.python.org/downloads/)
[PDM](https://pdm-project.org/en/latest/#installation)
[Git](https://git-scm.com/install/)

```bash
    git clone https://github.com/GildasLepennetier/opt-out-automation.git
    cd opt-out-automation
    pdm install
```
